# 🔌 PLUG.AI - Elite AI Career Agent

> **An Overpowered, GenAI-Native Career Co-Pilot.** > Plug.ai transforms the static job search into a dynamic, agentic workflow. Upload your resume, chat with the AI, and let it autonomously hunt, filter, and score live market openings based on your specific tech stack and experience.

---

## ✨ Features

* **Conversational Interface:** Forget static forms. Chat naturally with your agent using a sleek, Gemini-style UI built on Streamlit's chat elements.
* **GenAI Reasoning (The Brain):** Powered by Groq and `LLaMA-3.3-70B`, Plug.ai doesn't just do keyword searches. It understands your intent, parses unstructured PDF data, and generates optimized semantic queries.
* **Live Market Data:** Integrates directly with the JSearch API (via RapidAPI) to pull real-time job listings from across the globe.
* **Tech-Noir UI/UX:** A custom-styled, deep dark mode interface featuring glassmorphism cards, neon cyan circuit traces, and high-end typography.
* **Contextual Memory:** The agent remembers your session history, allowing you to seamlessly pivot from "React jobs in Lahore" to "What about remote roles in Dubai?" without losing your previous results.

---

## 🛠️ Tech Stack

* **Frontend:** [Streamlit](https://streamlit.io/) (with heavy Custom CSS injection)
* **LLM Engine:** [Groq](https://groq.com/) (LLaMA-3.3-70B-Versatile)
* **Data Source:** [JSearch API](https://rapidapi.com/letscrape-6bRBa3Q3O/api/jsearch)
* **Document Parsing:** `pdfplumber`
